Images

Images are inherently inaccessible by nature, particularly for those with visual disabilities.

Alternate text

Alternate texts can be used to include a description of an image. This will ensure that assistive technologies, such as screen readers, read this out to site visitors.

Long description

For images that require more complex description, such as a graph, or a chart, a more detailed description is usually required.

Aesthetic images

Images that serve no purpose other than for aesthetic reasons, such as decorative images, can safely be ignored by screen readers. This can be achieved by simple including an empty string as the value for the alt attribute.

Forms and form validation

Forms are an essential part of any website, as it is the platform on which visitors interact with the site. As such, accessibility plays an important role in ensuring the best possible experience for site visitors.
